This project has been developed using Vite, React, and TypeScript to ensure optimal 
performance and a smooth developer experience. To set up the project locally, 
please ensure that Node.js is installed on your machine. Clone the repository 
or download the project files, navigate to the project directory, 
and execute npm install to install all required dependencies. 
After the installation is complete, you can start the development server by running npm run dev. 
The application will be accessible at http://localhost:5173/. 
Key dependencies used in this project include react, react-dom, typescript, and vite, all of which are listed in the package.json file. 
The project uses custom CSS for styling to maintain flexibility and full control over the design. 
Detailed setup instructions, framework and language choices, and any important design decisions are documented in the README for seamless onboarding and usage.
